# Cryptocurrency Tracker
A comprehensive cryptocurrency tracking application that provides real-time data, market trends, and detailed analytics. Users can easily access information about their favorite coins, view live price changes, trading charts, and other market statistics. The app is highly interactive and allows users to fetch data for specific coins by appending the coin name to the URL.

## 🚀 Features
Live Price Updates: Get real-time price data for various cryptocurrencies.
Trading View Charts: Integrated TradingView charts for detailed analysis.
Trending Coins: Displays the most trending coins with line charts and 24-hour price changes.
Detailed Coin Information: Append a coin name in lowercase to the URL (e.g., /bitcoin) to view specific coin data.
Market Insights: Comprehensive market details for informed decision-making.
